# Next Rendering Demo

A minimal Next.js App Router project that showcases the major rendering modes:

- **SSG** – Static Site Generation
- **SSR** – Server-Side Rendering
- **CSR** – Client-Side Rendering
- **ISR** – Incremental Static Regeneration

## Getting Started

```bash
npm install
npm run dev
```

Visit http://localhost:3000 to explore each rendering mode. Adjust `NEXT_PUBLIC_BASE_URL` (or `VERCEL_URL` in deployment) if you proxy the API through a different origin.
